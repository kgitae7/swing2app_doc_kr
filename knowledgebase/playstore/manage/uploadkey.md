# 플레이스토어 업로드키 재설정 방법

***



앱을 플레이스토어(Play Store)에 등록하고 지속적으로 업데이트하려면 앱 서명과 업로드 키에 대한 이해가 꼭 필요합니다.

특히 업로드 키를 분실하거나 문제가 생겼을 때는 재설정 절차를 밟아야 앱 업데이트가 가능합니다.

구글 플레이스토어 업로드 키 재설정 방법에 대해 안내해드리겠습니다.

***



**✅플레이스토어 업로드 키 재설정이란 무엇이고, 왜 필요한가요?**

{% hint style="info" %}
**업로드 키(Upload Key)란?**

안드로이드 앱은 APK/AAB 파일을 반드시 서명(Signing)해야 구글 플레이에 올릴 수 있습니다.

Upload Key (업로드 키)

* 개발자가 플레이스토어에 앱을 업로드할 때 사용하는 키입니다.
* 일종의 “임시 서명” 같은 개념이고, 구글이 받은 뒤 최종적으로 App Signing Key로 다시 서명합니다.
* 따라서 보안상 업로드 키가 유출되거나 분실되더라도, 앱 자체의 최종 서명에는 문제가 없습니다.
{% endhint %}

{% hint style="success" %}
**업로드 키 재설정이 필요한 경우**

업로드 키는 개발자 PC에서 보관해야 하는데, 아래와 같은 상황이 생길 수 있습니다:

1.키스토어 파일 분실

업로드용 keystore(.jks) 파일을 잃어버린 경우

2.비밀번호 분실

keystore 또는 alias 비밀번호를 까먹은 경우

3.업로드 키 유출 의심

외부에 노출되거나 보안이 위협받은 경우

4.여러 개발자 간 키 충돌​

다른 개발자가 다른 키로 서명해서 업로드 오류가 나는 경우

이때는 기존 업로드 키로는 더 이상 앱을 업로드할 수 없으므로, 구글에 업로드 키 재설정을 요청해야 합니다.
{% endhint %}

{% hint style="warning" %}
**스윙투앱 고객사 안내**

스윙투앱 고객사의 경우, 외부에서 제작한 앱을 스윙투앱 서비스로 이전 업데이트할 때

기존 키스토어 파일이나 서명 정보를 모를 경우 업로드 키 재설정 절차를 활용할 수 있습니다.

스윙투앱에서는 고객분들이 필요한 PEM 파일 생성 및 제공을 도와드리니,

업로드 키 재설정이 필요하신 경우 언제든 고객센터로 문의해 주세요.
{% endhint %}

{% hint style="success" %}
해당 가이드의  구글 플레이 업로드키 재설정 방법은 구글 플레이에 앱서명이 되어 있을 경우만 셋팅이 가능합니다.&#x20;

앱 서명이 안되어 있다면 구글플레이에 인증서가 없으므로 키 재설정 진행이 불가합니다.&#x20;

**앱 서명 등록이 안되어 있는 분들은 해당 가이드대로 진행이 불가합니다.**&#x20;

Google Play 앱 서명에 등록되어 있지 않다면 안드로이드 키스토어 파일, 키스토어 패스워드, 키스토어 Alias, 키스토어 Alias 패스워드를 보내주셔야 합니다.&#x20;

그래야만 기존 앱 업데이트 진행이 가능합니다.&#x20;


{% endhint %}

## 업로드 키 재설정 진행하기

​

✅[**구글 플레이 콘솔**](https://play.google.com/console/developers) **접속 - 앱 선택**

{% embed url="https://play.google.com/console/developers" %}

<figure><img src="../../../.gitbook/assets/image (28).png" alt=""><figcaption></figcaption></figure>

업로드키 재설정할 앱 선택 후,

1\)왼쪽 메뉴: 테스트 및 출시 선택

2\)'앱 무결성' 선택

3\)Play 앱 서명:'설정' 선택

{% hint style="info" %}
**앱 서명이 되어 있을 경우만 설정에서 키 재설정 요청이 가능합니다.**&#x20;

앱 서명 하지 않았을 경우는 스윙투앱 고객센터로 따로 문의주시기 바랍니다.&#x20;
{% endhint %}

<figure><img src="../../../.gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure>

4\)'업로드 키 재설정 요청' 선택

​

<figure><img src="../../../.gitbook/assets/image (30).png" alt=""><figcaption></figcaption></figure>

5\)재설정 이유 선택: 아무 내용에 체크해도 무관합니다.

6\)4번 업로드 키 인증서에 생성된.PEM파일을 업로드하세요 \*.PEM파일을 올려주세요.

| <p><strong>스윙투앱 이용 사용자분들은 PEM파일을 만들어서 제공해드립니다.</strong></p><p><strong>따라서 스윙투앱 앱 이용할 예정 or 이용 중에 업로드 키 재설정이 필요하다면 스윙투앱 고객센터를 통해 문의주세요.</strong></p> |
| --------------------------------------------------------------------------------------------------------------------------------------------------- |

7\)요청 버튼만 누르면 완료됩니다.

​

<div align="left"><figure><img src="../../../.gitbook/assets/image (31).png" alt=""><figcaption></figcaption></figure></div>

업로드키 재설정은 2\~4일 정도 소요됩니다.

요청이 접수되었다는 메일이 들어오며, 메일에 언제부터 유효하다는 내용이 같이 기재되어 있습니다.

해당 날짜 이후 접속한 뒤, 새 버전의 앱을 업데이트 해주시면 되겠습니다.
