## 1. 앱 업데이트 하기

업데이트를 아직 안했다면 앱제작에서 업데이트를 먼저 해주세요.&#x20;

업데이트 신청을 위해서는 앱 새 버전 업데이트가 필요합니다.&#x20;

<figure><img src="../../../.gitbook/assets/image (146).png" alt=""><figcaption></figcaption></figure>

왼쪽 상단 **\[new 앱제작]** 선택

<figure><img src="../../../.gitbook/assets/image (147).png" alt=""><figcaption></figcaption></figure>

\[앱 업데이트] 버튼 선택

\*스토어 정책 업데이트로 인한 앱 업데이트 이거나 or 이번 업데이트가 사용자들에게 노출하지 않아도 될 경우라면, "업데이트 표시 안함"으로 체크 후 제작해주세요.

{% hint style="warning" %}
**웹뷰앱은 업데이트 팝업이 뜨지 않습니다.**

업데이트 버튼 선택시 바로 앱제작이 시작됩니다.

**Q.웹뷰앱은 왜 업데이트 표시 옵션 선택이 안뜨나요?**

웹뷰앱은 업데이트시 앱에서 업데이트 안내 팝업이 뜨지 압습니다.

따라서 업데이트 표시 옵션 자체를 선택할 필요가 없습니다.
{% endhint %}

{% hint style="success" %}

## 3. 플레이스토어 업로드 신청완료

<figure><img src="../../../.gitbook/assets/image (25).png" alt=""><figcaption></figcaption></figure>

**✅플레이스토어 업로드 신청이 들어오면 마켓업로드 지원팀 담당자가 확인하여 스토어에 등록해드립니다.**

**✅업데이트도 앱 심사가 다시 들어갑니다. 업데이트 심사에서 정책 위반으로 심사가 거절될 수도 있습니다.**

**거절될 경우 조치사항 안내드리며, 업데이트 승인이 완료될 때까지 추가 비용 없이 지원해드립니다.**&#x20;

**✅심사 기간은 신규 등록처럼 오래 걸리지 않으며, 업데이트 심사 1\~3일 소요됩니다.**&#x20;

**✅**&#xC5C5;로드 신청은 업무 시간내 신청 주시면 당일 내 바로 진행해드리나, 주말 및 업무시간 외에 업로드 신청을 할 경우 당일 심사 불가하며 평일 업무시간에 순차적으로 진행됩니다.

따라서 업로드 신청시 반드시 해당 내용 유념해주세요.

<figure><img src="../../../.gitbook/assets/구분선 (10).PNG" alt=""><figcaption></figcaption></figure>

## 4.앱스토어업로드 안내사항

{% hint style="warning" %}
<mark style="color:orange;">**안내사항**</mark>

1\. 플레이스토어 업로드 티켓은 당사에서 앱을 대신 올려드리는 대행비이며, 1회 20,000원입니다.

2\. 플레이스토어 - 신규 등록 심사는 1주일 정도 소요되며, 업데이트 심사는 1\~3일 정도 소요됩니다.&#x20;

충분한 시간을 두시고 기다려 주시기 바랍니다.

\*주말 및 업무시간 외에 업로드 신청을 할 경우 당일 심사 불가하며, 평일 업무시간에 순차적으로 진행되기 때문에 업로드 진행이 늦어질 수 있습니다.

3\. 심사가 거절되면 거절사유와 함께 심사에 필요한 내용을 추가적으로 더 요청할 수도 있습니다.

\*사용자분은 조치사항에 대해서 당사로 메일 회신주셔야 재심사 진행이 가능하며, 일주일 내로 회신을 주지 않으면 업로드 건은 등록 보류로 처리됩니다. (등록 보류 건은 이후 다시 연락주시면 재심사 진행해드립니다)

4\. 업로드가 완료되면 스토어 등록 완료 메일, 카톡 알림톡을 보내드립니다. (번호가 없을시 이메일만 발송됩니다.)

5\. 업데이트 된 경우, 플레이스토어에도 새 버전으로 다시 제출을 해야 합니다. \
그래야만 기존 사용자들이 업데이트를 받을 수 있고,신규 사용자들도 변경된 새 버전앱으로 설치가 가능합니다.&#x20;

6.플레이스토어는 사용자가 직접 업데이트도 가능합니다. 직접 하실 경우 티켓 비용 없습니다.

플레이스토어 업데이트 방법은 가이드 제공해드리고 있으며, 참고해주세요.

👉[플레이스토어 앱 업데이트 방법 가이드 ](https://documentation.swing2app.co.kr/knowledgebase/playstore/palystore-update)

7\. 구글 심사 중에도 유료앱 버전으로 심사가 들어가기 때문에, 앱 심사 기간 동안 이용하지 못한 이용일수에 대해서는 따로 일수를 추가해드리지 않습니다.

8\. 사용하지 않는 않은 티켓은 100% 환불 가능하지만, **업로드 신청으로 이미 사용을 했거나 심사가 거절되어 정상적으로 출시가 되지 않아도 이미 등록을 한 앱에 대해서는 티켓 환불이 어렵습니다.**

**9. 구글 플레이는 반드시 사용자분의 구글 개발자 계정을 만들어서 제공해주셔야 합니다.**&#x20;

&#x20;구글은 개발자 계정 대리 등록 불가합니다.&#x20;
{% endhint %}

<details>

<summary><strong>플레이스토어 업데이트도 심사가 다시 들어가나요?</strong></summary>

네, 업데이트도 앱 심사가 다시 들어갑니다.

따라서 앱스토어 업데이트에서도 다른 정책상의 문제로 심사가 거절될 수도 있습니다.&#x20;

거절이 될 경우 조치사항 안내드리며, 조치사항대로 수정하여 다시 앱 심사를 진행하여 업데이트 해드립니다.&#x20;
